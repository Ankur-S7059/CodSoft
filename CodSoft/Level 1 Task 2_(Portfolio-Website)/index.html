My Projects

Project 1 :

In this project, I've made a wirelessly bluetooth controlled robot car using Arduino as the microcontroller. Also, we are able to control the robot using our smartphone. The car is integrated with a bluetooth module for seamless wireless communication between the robot car and the bluetooth enabled devices.
Project 2 :

MATLAB is a powerful, high-level programming language. MATLAB app builder provides the power to build different apps, to represent our idea in a GUI-friendly manner. In this project, I've made a MATLAB app to calculate Simple Interest and Compound Interest on a given principal amount at specific Rate of Interest for a given period of time. The code to calculate the interest amount based on the given data is given in the 'CALCULATE' button section, which when clicked gives the desired output.
Project 3 :

In this project, I've developed a basic calculator that will perform arithmatic operation like addition, subtraction, division and multiplication. It can perform decimal operations. It will display 'Infinity' if we try to divide any number by zero. In case of any invalid expression, there won't be any output visible. There's a clear screen feature to clear the display screen anytime we want. Finally , the '=' button displays the desired output after evaluation.
